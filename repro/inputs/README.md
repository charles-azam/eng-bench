# inputs/ — everything you may use

These four files are the **complete, sanctioned input** for the task. Build your model from these
plus first-principles physics. They contain geometry, materials, sensor locations, and boundary
conditions — **no measured results**.

| File | Contents |
|---|---|
| `01_facility_geometry.md` | Facility description, scaling, and dimensions (cavity, plate, risers, plena, chimney) |
| `02_materials_and_properties.md` | Surface emissivities, steel & insulation properties, air properties |
| `03_instrumentation_map.md` | Where each quantity is measured (report your predictions at those locations) |
| `04_boundary_conditions_and_test_cases.md` | Heat-input and ambient conditions for each operating case |

See `../TASK.md` for the full brief and rules.
